# 🎲 Dice Roll Game in Python

A simple and interactive Dice Roll Game built using Python.  
This project demonstrates the use of:

- Functions
- Random module
- Menu-driven programming
- Modular code structure

The game allows users to play single-player, two-player, and multiple-round dice games.

---

## 🚀 Features

- 🎲 Random dice rolling using Python's `random` module
- 👤 Single Player Dice Roll
- 👥 Two Player Dice Game
- 🔁 Multiple Rounds Gameplay
- 🏆 Winner announcement after every game
- 📋 Menu-driven interface using `while True`
- 🧠 Clean and modular code structure

---

## 🧩 File Description

### `main.py`
Handles the menu system and user interaction.

### `game_service.py`
Contains all game logic such as:
- Single-player mode
- Two-player mode
- Multiple rounds mode

### `dice.py`
Generates random dice values using the `random` module.
